Guidelines

This Web Application was programmed by Ryoshi if you have any questions or something else to mention pls contact me via my GitHub

For now with this Web Application you can just store dogs and Owner in a Database. Via the dog Tab you can see all dogs and with a click on "Add a new Dog" you can add a new dog to the database. Every field is required even the owner. We do not accept dogs without an owner. To add an Owner you can visit the owner tab and click on "Add a new Owner". Like in the add dog tab every single field is required!
